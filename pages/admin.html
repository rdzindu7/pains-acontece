Sistema

Configurações

Conexão Supabase, sincronização automática e modo offline do portal.

Banco de Dados Supabase

Verificando conexão…

Com o Supabase configurado, notícias ficam na nuvem. O site sincroniza sozinho a cada 30 min via GitHub Actions (data/articles.json). Ao publicar, não precisa exportar manualmente.

  1. Crie um projeto grátis em supabase.com
  2. Execute o arquivo supabase/schema.sql no SQL Editor
  3. Em Settings → API, copie URL e anon key para js/core/config.js
  4. Em Authentication → Users, crie o admin: admin@painsacontece.com.br
  5. Faça commit no GitHub — o site atualiza automaticamente

Modo Offline (sem Supabase)

Se o Supabase não estiver configurado, as edições ficam no navegador. Exporte o JSON e envie para o GitHub manualmente.
